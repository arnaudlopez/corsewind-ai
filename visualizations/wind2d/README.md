# Wind2D viewer

Leaflet viewer for the current public CorseWind product:

- Esri World Imagery satellite basemap;
- AROME 10 m context layer over Corsica;
- optional WindNinja Corse 50 m tile overlays generated by the forecast engine;
- live color scale, layer toggles, hourly timeline, particle animation, and point speed/direction readout.

Generated payloads are intentionally not committed:

```text
visualizations/wind2d/arome-corsica-latest.json
visualizations/wind2d/windninja-corsica-tiles-50m/
visualizations/wind2d/windninja-corsica-data-50m/
```

Run the local gzip-capable static server from the repository root:

```bash
python3 scripts/serve_wind2d_compressed.py --port 8769
```

Then open:

```text
http://127.0.0.1:8769/visualizations/wind2d/
```

The page also works with `python3 -m http.server`. The raw model payloads are
loaded from generated `.json.gz` companions first, then fall back to plain
`.json` if needed.

To regenerate the data, run the forecast engine documented in `docs/forecast_update_engine.md`.
